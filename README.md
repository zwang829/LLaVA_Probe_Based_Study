# LLaVA Probe-Based Study

This repository contains the code for a probe-based study on LLaVA models. The core implementation is located in the `mywork/` directory.

## Environment Setup & Workspace Restoration

As this project heavily relies on the official model repositories, please follow the steps below to restore the complete directory structure before running any code:

1. Clone this repository:
   ```bash
   git clone https://github.com/zwang829/LLaVA_Probe_Based_Study.git
   cd LLaVA_Probe_Based_Study
   ```

2. Clone the official dependency repositories into the current directory:
   ```bash
   git clone https://github.com/haotian-liu/LLaVA.git
   git clone https://github.com/LLaVA-VL/LLaVA-RLHF.git
   ```
